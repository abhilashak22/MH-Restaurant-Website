# 🍴 Swaad Maharashtra - Responsive Restaurant Website

A fully responsive, multi-page restaurant website developed as part of the *Computer Networks Lab (Experiment 08)*.

## 🚀 Features
- *Responsive Design:* Works on Mobile, Tablet, and Desktop using CSS Media Queries.
- *Maharashtrian Menu:* A localized menu card featuring Thalis and classic dishes.
- *Table Reservation:* A functional booking form with *JavaScript Validation*.
- *Location Integration:* Embedded Google Maps for the Chandwad location.

## 🛠️ Technologies Used
- *HTML5* (Structure)
- *CSS3* (Styling & Responsiveness)
- *JavaScript* (Form Logic & Validation)
